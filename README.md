# Verilog Digital Design Projects

This repository contains a collection of **digital design projects implemented using Verilog HDL**. Each project demonstrates important hardware design concepts along with **synthesis using timing constraints and report generation (area, timing, and power)**.

## Projects Included

### 1. 8-bit Pipelined Multiplier
An **8-bit signed pipelined multiplier** that performs high-speed multiplication using pipeline stages to improve throughput and clock performance.

### 2. RAM Model
A **parameterized RAM module** supporting read and write operations using control signals and a bidirectional data bus.

### 3. Shift Register
A **parameterized shift register** capable of serial shifting, serial output, and parallel data loading.

### 4. Universal Asynchronous Receiver (UAR)
A **serial communication receiver** that converts incoming serial data into **8-bit parallel output** with ready and error flags.

### 5. Up Counter
A **4-bit asynchronous ripple up counter** implemented using T Flip-Flops that counts in binary sequence on each clock cycle.

## Tools Used
- Verilog HDL
- Synthesis tools with SDC constraints
- Simulation testbenches
- Hardware report generation (Area, Timing, Power)

## Purpose
These projects demonstrate **fundamental digital design concepts, modular hardware design, simulation, synthesis, and performance analysis** using Verilog.

## Applications
- FPGA and ASIC design
- Digital signal processing systems
- Embedded system hardware design
- Communication systems
